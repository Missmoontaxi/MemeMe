# Meme Me!

A super fun, responsive meme viewer with smart shuffle logic that auto-loads from GitHub and avoids repeats until you've seen them all.

🌐 **Live:** [welcometomemetown.com](https://www.welcometomemetown.com)

---

## What it does

- Come on down to Memetown!!!
- Responsive meme viewer that works across devices, best 5 minutes you'll have all day
- Smart shuffle — won't repeat a meme until you've cycled through every one
- Auto-loads images from a GitHub folder. Drop a new image in this repo, it shows up on the site. No CMS, no build step.

## Stack

- **Built with:** [Lovable](https://lovable.dev)
- **Image hosting:** GitHub (this repo)
- **Domain:** welcometomemetown.com

## Why this exists

Some side projects are research. This one's just for joy and a fun way to test what Lovable can do when you give it a clean product spec and a constrained scope.
